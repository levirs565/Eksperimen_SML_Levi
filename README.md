# Eksperimen SML

Untuk menjalankan proyek, disarankan menggunakan Poetry.

Install dependensi dengan Poetry:

```sh
poetry install --no-root  
```

Install dengan pip (jika tidak menggunakan Peotry):

```sh
pip -r requirements.txt
```

Untuk menjalankan notebook, pastikan memakai Virtualenv dari Poetry.

Untuk menjalankan skrip otomatisasi:

```sh
poetry run python ./preprocessing/automate_Levi.py
```
